# ShotGame-Portfolio
Photon을 이용한 1:1 FPS 멀티플레이 게임 프로젝트

## 주요 기능
- PhotonNetwork 기반 1:1 멀티플레이
- 로비, 방 기능
- PhotonView와 RPC를 통한 각 플레이어 동기화
- 1인칭 시점 : 컬링마스크로 내 모델을 숨김
- 1:1 결투 시작 시퀀스(시작연출)

## 기술 스택
- Unity 2021.3.37f1​
- Photon PUN2

## 실행 방법
- Unity 2021.3.37f1​에서 `Assets/Scenes/OpenScene.unity` 열고 실행
- 또는 빌드 후 실행
